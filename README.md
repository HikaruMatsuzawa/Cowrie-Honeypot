# Cowrie-Honeypot

AWS上で安全に運用する，CowrieベースのSSHハニーポット観測システムの文書リポジトリです．

本プロジェクトの目的は，攻撃機能を提供することではなく，外部から届くSSHブルートフォースやLinux/IoT機器を狙うボットの挙動を，低対話型ハニーポットで受動的に記録し，ログを分析することです．
